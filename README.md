This is single-source-of-truth (kinda like a bible) for a product. I love creating this for any new or existing product to have the foundational product info like-product narratives, ICPs, competition and market landscape- all in one place. This is a living document and gets udpated as the product and market evolve. This is an internal document that should be created together with the product teams and shared with the sales/go-to-market/customer success and any other customer facing teams. In general also works as a great internal enablement doc for anyone who wants to learn more about the product
